Kodsnack 159 - Korthuset vi byggt för att hålla ordning på korthuset

2016-06-07 05:26
Ladda ner (mp3)

Kristoffer och Fredrik rapporterar från Devsum 16 om kaffe och tankar från Tim Huckabys keynotepresentation. Teknikdemonstrationer kan leda till bra diskussioner kring utvecklaretik. Finns det någon aktiv rörelse kring etiska regler för utvecklare? Vi snackar också typer, Typescript och varför Typescript inte tilltalar Kristoffer. Vi kommer också in på Javascript och språkutveckling.

Ett stort tack till Cloudnet som sponsrar vår VPS!
